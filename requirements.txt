# ============================================================
# Sentinel — Python Dependencies
# ============================================================
# This project intentionally does NOT use a virtual environment.
# Packages that have an official Arch package are installed via
# pacman (not pip), since pip-installing over pacman packages on
# Arch can cause version conflicts and is generally discouraged.
#
# Step 1 — Install via pacman first:
#   sudo pacman -S python-scapy nmap python-pip python-pandas python-numpy
#
# Step 2 — Install the rest via pip:
#   pip install -r requirements.txt --break-system-packages
#
# (--break-system-packages is required on Arch when installing
#  system-wide without a venv — this is expected and safe here.)
#
# Note: Scapy requires root privileges to capture packets.
# Note: iptables commands require root privileges.
#       Run Sentinel with: sudo python main.py
# ============================================================
